
# e-learning-platform

## Fully functional web application written in EJS, Node.js, MongoDB.

### Features


**Instructors**<br>
_All changes made by teacher are updated on student dashboards_
* Create course
* Invite students
* Edit course description, name, members
* Delete course

**Students**
* Join and leave course
* View course pages
